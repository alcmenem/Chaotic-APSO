This is a CAPSO algorithm specifically adjusted to the optimization of electromagnetic cloaking of a core. It is meant to accompany a specific model which is not yet uploaded,
but can be provided.


Therefore, this code file is accompanied by many comments providing clarity regarding the complex optimization problem it has provided feasible solutions for. The code can be adjusted 
to work for any objective function, and the function of a sphere is provided, alongside basic initialization and boundary checking functions. However, the focus of this experiment is
not related to benchmark testing. If in search for an APSO demo you could check ref. [2].

The CAPSO algorithm is a proposed variant of the APSO algorithm, and it utilizes chaotic maps for parameter tuning, and useful randomness.
Reference:

[1] Gandomi, A.H.; Yun, G.J.; Yang, X.S.; Talatahari, S. Chaos-enhanced accelerated particle swarm optimization. Communications in Nonlinear Science and Numerical Simulation2013,18,
 327–340.

The code is built upon XS Yang's APSO demo available at (given the github upload date):
